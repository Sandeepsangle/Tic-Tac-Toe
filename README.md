# Tic-Tac-Toe Game

This is a simple Tkinter Tic-Tac-Toe game implemented in Python.

## Features

- Play against a computer opponent.
- Basic user interface.
- Logic to determine the winner or declare a draw.
- Option to restart the game.

## Requirements

- Python 3.x

   cd Tic-Tac-Toe
